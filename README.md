# PHP OOP User Management System

##  Project Description

This project is a simple User Management System developed using PHP and Object-Oriented Programming (OOP) concepts. It demonstrates how users can be created, authenticated, and managed using classes and services.

##  Features

* User registration and login system
* Admin and Regular User roles
* Authentication service
* Password hashing for security
* Clean OOP structure (Models, Services, Core)

##  Technologies Used

* PHP
* OOP (Object-Oriented Programming)
* Git & GitHub

##  Project Structure

* App/Core → Base classes and interfaces
* App/Models → User classes (Admin, RegularUser)
* App/Services → Authentication logic
* index.php → Main execution file

##  How to Run

1. Place project in XAMPP htdocs folder
2. Start Apache server
3. Open browser:
   http://localhost/user-management/index.php
